protocol in `METHODOLOGY.md`. Each session is a numbered markdown file using
`TEMPLATE.md`. The transcripts are edited for anonymization before publication
(no banks, counterparties, deals, or colleagues named — ever) but verdicts and
reasoning are otherwise unedited, including the OPEN items and the "I don't
remember" answers. Those are data, not embarrassments.
